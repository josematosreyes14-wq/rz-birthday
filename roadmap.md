# Filosofía del proyecto

No pretende ser Spotify.

No pretende ser un reproductor de música.

Pretende ser una forma diferente de recorrer una colección de canciones.

Cada canción debe sentirse como un pequeño momento, con su propia atmósfera, colores y personalidad.

# Notas para desarrollo

## Regla principal

Siempre priorizar:

Experiencia > Cantidad de funciones

## Antes de añadir una nueva función preguntarse:

- ¿Hace la experiencia más agradable?
- ¿Rompe la simplicidad del proyecto?
- ¿Puede hacerse reutilizable?
- ¿Mantiene el estilo minimalista?

Si la respuesta es sí, adelante.
